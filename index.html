Snake Game (Turtle)

The Snake Game is a classic arcade game recreated using Python's Turtle graphics library. In this game, the player controls a snake that continuously moves around the screen. The objective is to eat food items, which causes the snake to grow longer. The challenge increases as the snake grows, making it harder to navigate without colliding with the walls or its own tail. This version of the Snake Game combines simplicity with the nostalgic appeal of the original, providing an engaging coding experience and a fun challenge for players.

Tetris (PyGame)

Tetris is a timeless puzzle game brought to life using Python's Pygame library. Players must strategically arrange falling blocks, known as Tetriminos, to create complete horizontal lines. Each completed line is cleared from the screen, earning the player points. As the game progresses, the Tetriminos fall faster, requiring quick thinking and precision. The goal is to prevent the blocks from stacking up to the top of the screen, which would end the game. This version of Tetris captures the essence of the original while leveraging Pygame's capabilities to deliver smooth gameplay and an engaging challenge.
